# cloudsync-edge-secure-backup
Лёгковесный Python-агент, который следит за рабочей папкой в реальном времени и при любом изменении файла мгновенно создаёт его зашифрованную копию в целевом хранилище. Работает по событийной модели (через watchdog), без постоянного опроса диска — никаких циклов while True: scan_folder().
